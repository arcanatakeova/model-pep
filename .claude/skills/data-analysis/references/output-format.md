# Analysis Report Format

## Required Sections

1. **Executive Summary** (3-5 sentences)
   - What data was analyzed
   - Key finding headline
   - Primary recommendation

2. **Data Overview**
   - Source file(s) and format
   - Shape (rows x columns)
   - Data quality (missing values, duplicates, anomalies)

3. **Key Findings** (numbered, with supporting statistics)
   - Each finding includes the metric, value, and interpretation
   - Include confidence intervals where applicable

4. **Methodology Notes**
   - Tests used and why
   - Assumptions checked and results
   - Limitations of the analysis

5. **Recommendations** (actionable next steps)
   - Prioritized by impact
   - Include data-driven justification

6. **Appendix** (full statistical output)
   - Complete summary statistics table
   - All test results with p-values
   - Visualizations if generated
