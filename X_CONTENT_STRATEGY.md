## Content Pillars (What to Post)

### Pillar 1 — "Build With Me" Tutorial Threads (40% of content)
The main growth engine. Step-by-step threads with screenshots/photos at each step.

**Thread templates:**

1. **"I built a bot that trades crypto 24/7. Here's exactly how it works (thread):"**
   - Screenshot of the architecture diagram
   - Screenshot of terminal running `--scan`
   - Screenshot of equity curve going up
   - Screenshot of config.py with key params highlighted
   - CTA: "Want the full workflow? Link in bio"

2. **"How to set up leveraged crypto trading with Python in 10 minutes:"**
   - Photo: code editor with executor.py open
   - Photo: Binance API key creation screen
   - Photo: terminal showing paper trade results
   - Photo: risk_manager.py safety features highlighted
   - CTA: "Get the complete setup workflow → [link]"

3. **"I snipe Solana memecoins automatically. Here's the system:"**
   - Screenshot: DEX Screener scanning
   - Screenshot: Jupiter swap execution
   - Screenshot: token_safety.py rug-check logic
   - Screenshot: P&L from DEX trades
   - CTA: "Full Solana sniper workflow → [link]"

4. **"6 indicators combined into 1 signal. Here's my ensemble strategy:"**
   - Photo: each indicator explained (RSI, MACD, Bollinger, etc.)
   - Photo: ensemble.py scoring logic
   - Photo: signal strength → leverage mapping table
   - Photo: backtest results
   - CTA: sell the strategy config as a workflow

### Pillar 2 — Results & Proof Posts (25% of content)
Screenshots of real results build trust and sell without selling.

- Daily/weekly equity curve screenshots
- "Bot found this trade at 3am while I slept" posts
- Before/after portfolio screenshots
- Comparison: manual trading vs bot performance
- Polymarket prediction wins

### Pillar 3 — Quick Tips & Alpha (20% of content)
Single tweets, not threads. High engagement, easy to produce.

- "One line of Python that saved me from a rug pull:" + screenshot of token_safety.py
- "The leverage setting most people get wrong:" + config screenshot
- "Stop loss at 3% vs 5% — here's what the data shows:" + backtest chart
- Hot takes on AI + trading intersection
- Repost/quote-tweet trending crypto moments with your bot's take

### Pillar 4 — Behind the Scenes (15% of content)
Humanizes the brand, builds parasocial connection.

- Photos of your setup (monitors, terminal running)
- "Just pushed an update that adds [feature]" + code diff screenshot
- Bug stories: "My bot almost did X, here's how the safety caught it"
- Revenue milestone posts (workflow sales screenshots, blurred appropriately)

## Thread Photo/Screenshot Playbook

Every thread MUST include visuals. Here's how to make them:

### Screenshot Types That Convert
1. **Code screenshots** — Use [ray.so](https://ray.so) or Carbon for styled code blocks
2. **Terminal output** — Clean terminal with results highlighted (use colored output)
3. **Charts/graphs** — Equity curves, backtest results, signal visualizations
4. **Architecture diagrams** — Draw.io or Excalidraw for system overview
5. **Config comparisons** — Side-by-side "default vs optimized" settings
6. **Dashboard screenshots** — The bot's dashboard.html with real data

### Photo Rules
- First image in thread = the hook (most eye-catching result or diagram)
- Annotate screenshots with arrows/highlights (use Skitch or built-in markup)
- Consistent color scheme across all visuals (brand recognition)
- Include your X handle as watermark on screenshots
- 4 images per thread minimum, spread across key tweets in the thread

## Growth Tactics

### Phase 1: 0–1K followers (Weeks 1–6)
- Post 2 high-quality threads per week
- Reply to every large AI/crypto account's posts with genuine insights
- Quote-tweet trending topics with your bot's perspective
- Join Twitter Spaces about AI trading and speak
- DM 5 relevant accounts per day for genuine conversation (not spam)

### Phase 2: 1K–5K followers (Weeks 6–16)
- Increase to 3 threads per week
- Start cross-posting thread summaries to Reddit (r/algotrading, r/cryptocurrency)
- Launch first workflow product, announce with a thread
- Collect testimonials from early buyers, screenshot and post
- Collaborate with other AI/trading creators (co-threads)

### Phase 3: 5K–25K followers (Months 4–8)
- Launch subscription product
- Create a free Discord/Telegram for community (funnels to paid)
- Start affiliate partnerships (Binance referral, VPS providers)
- Consider short-form video (thread content repurposed to Reels/TikTok)
- Hire a VA to handle DMs and basic support
